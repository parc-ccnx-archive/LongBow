10  * Define a test case with the given `fixtureName` and `testCaseName` with an explictly specified status.
11

13  * The `testCaseName` is the name of the Test Case and must be valid syntax for a C identifier.

16

18  * @code
19  * LONGBOW_TEST_CASE_EXPECTS(MyFixture, alwaysWarn, .event = &LongBowAssertEvent)
20  *

If the termination status of the test must is equal to the specified status, the test will be signaled as `LONGBOW_STATUS_SUCCESS, otherwise it will be LONGBOW_STATUS_FAILED.
