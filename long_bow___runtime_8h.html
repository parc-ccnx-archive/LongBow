void longBowRuntime_Abort ( void  )

Abort the running process using the current runtime environment. If the configuration is set to produce a core dump, this function simply returns. This permits the caller to use the form: longBowRuntime_Abort(), kill(0, SIGTRACE) To generate a core image.

See the assertion macros for how this is used.
